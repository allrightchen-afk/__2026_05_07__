## 💎 核心特點 (Core Features)

- 🚀 **雙軌獨立運作**：專為「甲、乙」兩隊設計的獨立計時面板，支持同步操作與各自暫停/重設。
- 🎨 **現代化視覺設計**：基於毛玻璃效果與大圓角語彙，營造極具質感、清晰易讀的競賽氛圍。
- ⚡ **毫秒級精確響應**：採用高效 React 狀態管理，確保每一次點擊都能即時反應，杜絕競賽爭議。
- 📱 **全平台適配**：響應式佈局設計，無論是在禮堂的大型投影幕還是裁判手持的平板上都能完美呈現。
- ✨ **流暢微交互**：內建豐富的動態回饋（Motion Animations），讓計時過程不再單調。

## 🛠️ 技術架構 (Technology Stack)

本專案採用目前最前沿的前端技術棧，確保高效能與易維護性：

- **Core**: [React 19](https://react.dev/) - 利用最新的 Hooks 與並行渲染技術。
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) - 打造高度定制化的 UI 組件。
- **Animation**: [Motion](https://motion.dev/) - 驅動流暢的物理動畫與組件轉場。
- **Icons**: [Lucide React](https://lucide.dev/) - 提供精緻且一致的向量圖標。
- **Build Tool**: [Vite](https://vitejs.dev/) - 提供極速的開發與建置體驗。

## ⚙️ 快速上手 (Quick Start)

### 1. 複製倉庫
```bash
git clone https://github.com/your-username/dual-track-timer.git
cd dual-track-timer
```

### 2. 安裝依賴
```bash
npm install
```

### 3. 開啟開發模式
```bash
npm run dev
```
啟動後請訪問 `http://localhost:5173`。
